# grc-compliance-dashboard# GRC Compliance Dashboard

## Overview

This project simulates an enterprise Governance, Risk, and Compliance (GRC) analytics dashboard using Python.

The dashboard analyzes:
- Compliance controls
- Risk scores
- Audit findings
- Compliance trends
- Department-level risk exposure

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Faker

---

## Features

- Synthetic enterprise dataset generation
- Risk analytics
- KPI calculations
- Compliance trend analysis
- Department risk scoring
- Executive dashboard visualizations
